Elixir Projects - Built with ❤️.

## Features
- Various Elixir projects
- Demonstrates Elixir capabilities

## Technology Stack
- Elixir
- Phoenix

## Usage
1. Clone the repository: `git clone https://github.com/rubbaliedo/elixir-projects.git`
2. Install dependencies: `mix deps.get`
3. Run the application: `mix phx.server`
